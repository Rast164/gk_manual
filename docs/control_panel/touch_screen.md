# Сенсорный экран
Сенсорный экран является средством вывода текстовой и символьной информации и основным органом управления ГК. Доступ к сенсорному экрану ограничен [профилями доступа пользователей]({% link docs/getting_started/getting_started.md %}#профили-доступа-пользователей).

</p>

### Индикатор состояния
Обобщенный индикатор состояния сенсорного экрана показывает текущее состояние прибора. Состояние отображается исходя из приоритета «Пуск» –- «Пожар» – «Внимание» – «Неисправность». Состояние «Пуск» имеет наивысший приоритет отображения – при наличии хотя бы одного запущенного устройства пожаротушения, индикатор будет отображать состояние «Пуск» даже при наличии сигнала «Пожар», «Внимание» или «Неисправность».
